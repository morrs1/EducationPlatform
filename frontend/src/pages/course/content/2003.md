# Git для начинающих

Курс помогает спокойно войти в Git без ощущения, что каждая команда может сломать проект. Мы идем от локальной истории и базовых коммитов к веткам, совместной работе и восстановлению после ошибок.

## Для кого этот курс

- Для начинающих разработчиков, которые только начинают работать с репозиториями
- Для студентов, которым нужно перестать путаться в merge, rebase и stash
- Для тех, кто хочет уверенно работать с Git в команде

## Что вы разберете

- создание и структура репозитория
- чистые коммиты и работа с историей
- ветвление, pull request и командный workflow
- восстановление после типичных ошибок и конфликтов

## Формат обучения

Сначала вы разбираете инструмент на простых бытовых сценариях, а потом закрепляете его на практических мини-кейсах. В конце курса у вас остается рабочий набор привычек, которые действительно помогают в реальном проекте.
