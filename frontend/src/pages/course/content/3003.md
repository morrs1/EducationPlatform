# React.js для начинающих

Этот курс собран как вход в современный React через живые интерфейсные сценарии. Вместо абстрактных примеров мы последовательно строим UI, работаем с состоянием, роутингом и постепенной декомпозицией приложения.

## Для кого этот курс

- Для разработчиков, которые знают JavaScript и хотят перейти к React
- Для тех, кто хочет понять не только синтаксис, но и архитектурную логику UI
- Для студентов, которым важен путь от простых компонентов к целой странице

## Что будет внутри

- JSX, компоненты и композиция интерфейсов
- состояние, события и формы
- эффекты, загрузка данных и обработка ошибок
- разбиение приложения на page, widget, feature и entity

## Как вы будете учиться

Каждый модуль заканчивается маленьким прикладным результатом: от первого экрана до связанного mock-driven интерфейса. За счет этого курс помогает не только запомнить API React, но и начать собирать более чистую архитектуру проекта.
