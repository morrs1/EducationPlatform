# Что такое Python

Python — это язык программирования с простым и читаемым синтаксисом. Его часто выбирают для обучения, автоматизации, анализа данных, веб-разработки и написания небольших скриптов.

## Почему Python удобен для старта

У Python достаточно короткий и понятный код. Даже первая программа выглядит очень просто:

```python
print("Привет, мир!")
```

Эта строка говорит программе: выведи в консоль текст `Привет, мир!`.

## Что делает `print()`

Функция `print()` показывает результат работы программы пользователю.

Примеры:

```python
print("Привет!")
print("Python")
print("123")
```

## Что важно запомнить

- текст в Python записывается в кавычках
- `print()` выводит значение на экран
- программа выполняет инструкции сверху вниз

## Мини-проверка себя

Попробуй ответить мысленно на вопрос:

Что выведет программа?

```python
print("Добро пожаловать!")
```

Если ты понимаешь, что на экран попадет текст `Добро пожаловать!`, значит основная идея уже ясна.
