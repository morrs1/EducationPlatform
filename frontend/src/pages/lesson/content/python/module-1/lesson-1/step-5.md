# Приветствие пользователя

Напишите программу, которая:

1. считывает имя пользователя из стандартного ввода
2. выводит строку вида:

```text
Привет, <имя>!
```

## Пример

Ввод:

```text
Анна
```

Вывод:

```text
Привет, Анна!
```

## Подсказка

В Python ввод обычно получают через `input()`:

```python
name = input()
```

А затем используют в `print()`.

## Обрати внимание

- в ответе должна быть именно строка `Привет, ...!`
- после запятой нужен пробел
- в конце нужен восклицательный знак
