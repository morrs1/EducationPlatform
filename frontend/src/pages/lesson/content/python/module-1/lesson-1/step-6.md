# Сумма двух чисел

Напишите программу, которая:

1. считывает два целых числа
2. выводит их сумму

## Пример

Ввод:

```text
2
3
```

Вывод:

```text
5
```

## Подсказка

Если читать числа через `input()`, то по умолчанию они приходят как строки.

Чтобы превратить строку в число, можно использовать `int()`:

```python
a = int(input())
b = int(input())
```

После этого числа можно сложить и вывести результат:

```python
print(a + b)
```

## Важно

- программа должна считать два отдельных значения
- результат должен быть только один — сумма этих чисел
