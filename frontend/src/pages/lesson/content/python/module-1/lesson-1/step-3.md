# Что выведет программа

Посмотри на код:

```python
print("Привет")
```

Выбери правильный вариант ответа.

## Подсказка

В этом примере используется уже знакомая функция `print()`.

Она выводит на экран то значение, которое передано в круглых скобках.

Обрати внимание:

- значение записано в кавычках
- это строка
- строка будет показана пользователю без дополнительных изменений
