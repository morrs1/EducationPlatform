# Введите тип значения

Рассмотри значение:

```python
"hello"
```

Введите тип этого значения в Python.

## Что нужно вспомнить

Мы уже обсуждали, что текст в Python записывается в кавычках.

Например:

```python
"Привет"
"Python"
"hello"
```

Все такие значения относятся к одному и тому же типу.

## Формат ответа

Введите короткое название типа так, как оно используется в Python.
