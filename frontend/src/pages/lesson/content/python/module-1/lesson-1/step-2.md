# Переменные и строки

Переменная позволяет сохранить значение под именем, чтобы потом использовать его в программе.

## Пример переменной

```python
name = "Баграт"
print(name)
```

Здесь:

- `name` — имя переменной
- `"Баграт"` — значение
- `print(name)` — вывод переменной на экран

## Как работает присваивание

Оператор `=` в Python используется для записи значения в переменную.

```python
city = "Москва"
age = 25
```

Теперь программа знает, что:

- `city` содержит строку `"Москва"`
- `age` содержит число `25`

## Строки

Строка — это текстовое значение. Обычно строки записывают в кавычках:

```python
"hello"
"Привет"
"Python"
```

## Использование переменных в выводе

```python
language = "Python"
print("Я изучаю", language)
```

На экран будет выведено:

```text
Я изучаю Python
```

## Что важно запомнить

- переменная хранит значение
- строка — это текст
- переменную можно передать в `print()`
- имя переменной придумывает разработчик
